## Quick Start

1. **Phase 0 (Now)**: Start with `PHASE_0_GROUNDWORK/tracking-spreadsheet-template.csv`
2. **Email Outreach**: Use templates in `EMAIL_TEMPLATES/outreach/`
3. **Lead Tracking**: Import leads into `LEAD_GENERATION/target-lists/`
4. **Weekly Review**: Follow `MASTER_PLAYBOOK/weekly-operating-rhythm.md`

## Core Principle

Every tool here is designed to be **immediately actionable**. No theory—only systems you can use today.
